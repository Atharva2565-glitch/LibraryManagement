# 📚 Library Management System

A **Java Console-Based Library Management System** developed using **Object-Oriented Programming (OOP)** concepts. This project allows users to manage library books efficiently through a simple menu-driven interface. It supports adding, searching, issuing, returning, and storing book records permanently using file handling.

---

# 📖 Project Description

The Library Management System is a console application built in Java to simplify library operations. Instead of maintaining records manually, this application stores all book information digitally in a text file.

The project demonstrates the use of Java programming concepts such as classes, objects, collections, file handling, exception handling, loops, and conditional statements.

* 📚 Add New Books

# 📂 Data Storage

The project uses **books.txt** to store all book records.

Each record contains:

* Book ID
* Book Name
* Author Name
* Language
* Issue Status

Data is automatically loaded whenever the program starts and saved whenever changes are made.
